# 📝 Exam Portal

A web-based **Exam Portal** developed using **Flask, HTML, CSS, JavaScript, and SQLite**. The application provides a simple and interactive platform for managing online examinations with a clean, responsive user interface.

## 🚀 Features

- 👨‍🎓 Student Registration & Login
- 🔐 Secure Authentication
- 📝 Online Exam Interface
- 📊 Score/Result Display
- 📱 Responsive Design
- 🎨 Clean and User-Friendly UI
- ⚡ Flask Backend with SQLite Database

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Chetanyadavannavar/exam_portal.git
```

### Navigate to the Project

```bash
cd exam_portal
```

### Create a Virtual Environment

```bash
python -m venv myenv
```

### Activate the Virtual Environment

#### Windows

```bash
myenv\Scripts\activate
```

#### Linux/macOS

```bash
source myenv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000/
```
